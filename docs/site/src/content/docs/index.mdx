**Reuse production code** — [one decorator swap](/agentcore-rl-toolkit/guides/agent-adaptation/); no custom model wrappers.

**Isolated, auto-scaling rollouts** — every rollout runs in its own ACR microVM; thousands in parallel without contending for your machine.

**Decoupled environments** — agent container ships its own dependencies, fully separate from the training library.

**Training-ready** — **token IDs, logits, and expert masks** captured via `rllm-model-gateway`; results via S3.

New here? Start with the [math agent example](/agentcore-rl-toolkit/examples/strands-math-agent/)
